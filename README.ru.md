# ✋ Palmeter

**Palmeter** это инструмент для отслеживания рук в реальном времени на Python, который обнаруживает несколько рук и вычисляет **расстояние между ладонями**.

![PalmmeterДемо](https://github.com/KrishBharadwaj5678/Palmeter/raw/main/PalmeterDemo.png)

## 🌟 Возможности

| Функция                             | Описание                                                         |
| ----------------------------------- | ---------------------------------------------------------------- |
| ✋ Обнаружение нескольких рук        | Обнаруживает **две руки** в реальном времени с помощью MediaPipe |
| 📏 Расчёт расстояния между ладонями | Вычисляет **расстояние между ладонями**                          |
| 🎯 Отображение в реальном времени   | Динамически показывает расстояние на экране                      |
| 🖼️ Визуальные наложения            | Отображает ключевые точки рук для лучшего понимания              |
| 🔁 Непрерывное отслеживание         | Постоянно обновляет положение рук кадр за кадром                 |
| ⚡ Лёгкая производительность         | Быстро работает даже на слабых ПК, используя только CPU          |
| 🧪 Простота расширения              | Чистый код, готовый для добавления распознавания жестов          |
