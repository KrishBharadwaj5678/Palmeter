# ✋ Palmeter

**Palmeter** é uma ferramenta de rastreamento de mãos em tempo real desenvolvida em Python que detecta múltiplas mãos e calcula a **distância entre elas**.

![PalmeterDemo](https://github.com/KrishBharadwaj5678/Palmeter/raw/main/PalmeterDemo.png)

## 🌟 Funcionalidades

| Funcionalidade                     | Descrição                                                      |
| ---------------------------------- | -------------------------------------------------------------- |
| ✋ Detecção de Múltiplas Mãos       | Detecta **duas mãos** em tempo real usando MediaPipe           |
| 📏 Cálculo da Distância das Palmas | Calcula a **distância entre as palmas das mãos**               |
| 🎯 Exibição em Tempo Real          | Mostra dinamicamente a distância na tela                       |
| 🖼️ Sobreposições Visuais          | Desenha pontos de referência das mãos para melhor compreensão  |
| 🔁 Rastreamento Contínuo           | Atualiza continuamente as posições das mãos quadro a quadro    |
| ⚡ Desempenho Leve                  | Funciona rapidamente até em máquinas simples usando apenas CPU |
| 🧪 Fácil de Expandir               | Código limpo pronto para reconhecimento de gestos              |

---

## 🛠️ Tecnologias Utilizadas

| Tecnologia    | Descrição                                       |
| ------------- | ----------------------------------------------- |
| 🧠 MediaPipe  | Detecção de pontos das mãos                     |
| 🎥 OpenCV     | Biblioteca de visão computacional em tempo real |
| ⚙️ Zona CV     | Simplifica o rastreamento das mãos              |
| 🐍 Python 3.x  | Linguagem de programação                        |
